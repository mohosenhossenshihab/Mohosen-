# Mohosen-
E-commerce order management and operations tracking system.
Packly Ops

An e-commerce order management and operations system built for tracking orders, merchants, agents, and couriers.

## What it does

- Tracks orders through a 7-step status workflow
- Manages merchants, agents, and couriers
- Kanban board for visual order tracking
- Agent leaderboard for performance monitoring
- Duplicate order detection
- Automated workflows using Google Apps Script

## How Claude AI is used

This project uses Claude AI for:
- Building and debugging Google Apps Script automations
- Designing the admin panel UI logic
- Fuzzy product name matching algorithms
- Workflow optimization and automation planning

## Tech Stack

- Google Sheets (database)
- Google Apps Script (automation)
- HTML/CSS/JavaScript (admin panel)
- Claude AI (development assistant)

## Features

- Order status tracking (Pending → Processing → Shipped → Delivered)
- Merchant management
- Agent performance tracking
- Courier coordination
- Duplicate detection system
- Operations dashboard
